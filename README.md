<h1>Gulp Chrome Extensions</h1>
<p>The basic structure to start the development of Chrome Extensions like a boss :D</p>
<p>
  Gulp tasks like:
  <ul>
    <li>Browserify (gulp browserify)</li>
    <li>Compile Sass/Compass (gulp styles)</li>
    <li>Minify (gulp build)</li>
    <li>Watch modifications (gulp watch)</li>
  </ul>
</p>
<p>
  Clone the repository and:
</p>
<pre>
  <code>npm install</code>
</pre>
<p>
  You will need of gulp installed globally, if you don't have a gulp globally, execute the code:
</p>
<pre>
  <code>npm install -g gulp</code>
</pre>
<p>
  For run the gulp tasks, execute the commands:
</p>
<pre>
  <code>gulp browserify</code>
  <code>gulp styles</code>
  <code>gulp build</code>
  <code>gulp watch</code>
</pre>
<p>More information about how make a chrome extensions:</p>
<p><a href="https://developer.chrome.com/extensions/getstarted">https://developer.chrome.com/extensions/getstarted</a></p>
